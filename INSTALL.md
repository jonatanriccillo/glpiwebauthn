# Instalación

La guía completa está en **[doc/instalacion.md](doc/instalacion.md)**.

Resumen:

```bash
composer install --no-dev --optimize-autoloader
```

Copiá el plugin en `plugins/webauthn`, instalalo desde GLPI y configurá HTTPS + RP ID.

Sitio del proyecto: [https://github.com/jonatanriccillo/glpiwebauthn](https://github.com/jonatanriccillo/glpiwebauthn)
